Bom dia PessoALL,

O projeto consiste em um sistema com 3 tabelas que deve fazer as operações básicas, são elas:

- Cadastro
- Consulta
- Alteração
- Exclusão

Estas tabelas devem conter no minimo 3 atributos, elas nao precisam estar ligadas com FK, o script sql deve estar contido no projeto o arquivo de ant (build.xml) deve estar criando as tabelas no banco.

- Deve ser criado as operações para as 3 entidades, é necessário a criação de um menu para acessar as funcionalidades de cada uma.

- Deve ser usado também o CSS para personalizar o sistema, (Sugestao: www.dotemplate.com)

o projeto será individual e para entrega no dia 03/05/2009
